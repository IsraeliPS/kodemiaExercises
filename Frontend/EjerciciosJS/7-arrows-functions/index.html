Funciones Flecha

Este ejercicio consta de 15 problemas que son:

Ejercicio 1:
Dado un string de palabras, devolver la palabra más larga.

Ejercicio 2:
Crear un programa que permita detectar si una frase finaliza con punto.

Ejercicio 3:
Crear un programa que permita al usuario retornar el numero de coincideas de una palabra en una frase que el mismo usuario introduzca

Ejercicio 4:
Crear un programa que permita al usuario extraer una subcadena de una cadena dada.

Ejercicio 5:
Crear un programa que permita comparar dos palabras dadas por el usuario y determine si son iguales o no.

Ejercicio 6:
Crear un programa que permita a una cadena de texto muestre el caracter '-' entre cada caracter de la cadena.

Ejercicio 7:
Sabiendo que una cadena de texto contiene parentesis. Crear un programa que me devuelva la cadena que se encuetra entre los parentesis.

Ejercicio 8:
Realizar un programa que muestre una piramide del 1 al 9 de las siguientes formas:

Forma 1:     Forma 2:
1                 999999999
22               88888888
333             7777777
4444           666666
55555         55555
666666       4444
7777777     333
88888888   22
999999999 1

Ejercicio 9:
Solicitar al usuario un nombre por medio del promt e ingresarlo a una coleccion , validando que no se repita ningun nombre.

Ejercicio 10:
Dado una coleccion de numeros. obtener el elemento Mayor, Menor, el Promedio y la Suma de los elementos.

Ejercicio 11:
Solicitar al usuario por medio de un prompt una cadena de numeros separados por algun caracter y que cada numero se agregue a una colecion de numeros.

Ejercicio 12:
Dado un array de números, determinar por cada elemento si es un multiplo de 4, 3, 5, 2.

Ejercicio 13:
Crear un programa que dado un array de strings y un número, debe de imprimir cada uno de los items del array con los caracteres limitados al número ingresado.

Ejercicio 14:
Crear un programa que dados los datos de 3 lados de un triangulo con base en esos datos, determinar el área del triangulo y mostrarla en otro mensaje, indicar si el triangulo es isoseles, equilátero o escaleno.

Ejercicio 15:
Crear un programa que genere un string con "n" caracteres aleatorios, con la unica condicion de que no haya caracteres repetidos en el string.
